impulse.Scenes

Client-side scene system for playing cinematic sequences with camera control and optional text

Functions

impulse.Scenes.Play(stage, sceneData, onDone, skipPVS, preLoad)

 View source »

Plays a single cinematic scene.

Parameters

  • stage int

    Stage number of the scene

  • sceneData table

    Table containing scene configuration

  • onDone function optional

    Callback called when scene ends

  • skipPVS boolean optional

    If true, skips sending PVS update to server

  • preLoad function optional

    Callback called shortly before the scene ends (used to preload next scene)

impulse.Scenes.PlaySet(set, music, onDone)

 View source »

Plays a set of scenes in sequence with optional music

Parameters

  • set table

    A table of sceneData tables

  • music string optional

    Either a sound path or a web link

  • onDone function optional

    Called when all scenes in the set are complete
